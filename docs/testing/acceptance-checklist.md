# 验收清单

## 1. 功能验收

- [ ] 普通用户可以成功登录
- [ ] 用户可以提交校园认证申请
- [ ] 管理员可以审批认证申请
- [ ] 用户可以发布失物信息
- [ ] 用户可以发布招领信息并设置验证问题
- [ ] 系统可以展示搜索结果和筛选结果
- [ ] 系统可以生成匹配结果列表
- [ ] 用户可以发起认领申请
- [ ] 拾获者或管理员可以审核认领
- [ ] 双方可以确认交接
- [ ] 交接完成后状态可以自动关闭
- [ ] 系统可以发送站内消息
- [ ] 管理后台可以处理举报和查看基础统计

## 2. 数据与状态验收

- [ ] 失物状态流转正确
- [ ] 招领状态流转正确
- [ ] 认领状态流转正确
- [ ] 交接完成后生成交接记录
- [ ] 交接完成后产生积分流水

## 3. 安全与权限验收

- [ ] 未登录用户无法访问受保护资源
- [ ] 普通用户无法访问管理员接口
- [ ] 敏感物品信息不对外暴露原图
- [ ] 管理员关键操作有日志记录

## 4. 文档与交付验收

- [ ] 接口文档已更新
- [ ] 数据库设计文档已更新
- [ ] 测试用例已执行并有结果记录
- [ ] 部署说明可供他人复现
- [ ] 演示账号和演示数据准备完成
