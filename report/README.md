## 构建

需要 XeLaTeX、`ctex`、`tikz`、`booktabs`、`longtable`、`tabularx` 和系统字体 `Droid Sans Fallback`。在仓库根目录执行：

```bash
bash report/build-all.sh
```

脚本对每份报告执行三次 XeLaTeX，以稳定生成目录和交叉引用。构建日志与中间文件由根目录 `.gitignore` 排除，最终 PDF 纳入版本控制。

构建结束后，脚本还会把报告中的 12 张 TikZ 设计图导出到 `diagram-previews/`。这些 PNG 用于逐图检查；最终 PDF 继续使用矢量图以保证打印质量。也可以单独执行：

```bash
bash report/export-diagrams.sh
```
